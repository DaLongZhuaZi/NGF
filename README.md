## и®ҫи®ЎжЁЎејҸ

### з»ҹдёҖеҲҶеұӮжЁЎејҸ
жҜҸеұӮйҒөеҫӘ `contracts/` + `facades/` + `index.ets` з»“жһ„пјҡ
- **contracts/** вҖ” зәҜжҺҘеҸЈгҖҒжһҡдёҫгҖҒж•°жҚ®зұ»пјҲйӣ¶дҫқиө–пјү
- **facades/** вҖ” й»ҳи®Өе®һзҺ°пјҢз»§жүҝ `NGFIntegrationFacadeBase`
- **index.ets** вҖ” е…¬ејҖ API еҜјеҮә

### йӣҶжҲҗй—ЁйқўжЁЎејҸ
жҜҸдёӘйӣҶжҲҗй—Ёйқўз»ҹдёҖе®һзҺ°дёүжӯҘз”ҹе‘Ҫе‘Ёжңҹпјҡ
1. `getServiceRegistrations()` вҖ” еЈ°жҳҺжңҚеҠЎжіЁеҶҢйЎ№
2. `bootstrap()` вҖ” иҮӘеҠЁжіЁеҶҢеҲ° DI е®№еҷЁ + ServiceContainer
3. `syncAppStorage()` вҖ” еҗҢжӯҘзҠ¶жҖҒеҲ° AppStorage

### жЁЎеқ—еҗҜеҠЁжЁЎејҸ
`ModuleBootstrapCoordinator` з®ЎзҗҶжЁЎеқ—дҫқиө–еӣҫпјҢжҢүжӢ“жү‘жҺ’еәҸиҮӘеҠЁеј•еҜјгҖӮжЁЎеқ—йҖҡиҝҮ `NGFModuleDefinition` еЈ°жҳҺдҫқиө–е’ҢжңҚеҠЎд»ӨзүҢпјҢеҗҜеҠЁз»“жһңйҖҡиҝҮдәӢд»¶жҖ»зәҝе№ҝж’ӯгҖӮ
